# Plan: Arreglar Deployment de Vercel para Proyecto Lupa

## Problema Actual
El deployment en Vercel falla porque:
1. Error "next: command not found" - npm install no se ejecutó correctamente
2. Error "Unexpected non-whitespace character after JSON at position 194" - archivo package.json corrupto en Vercel
3. La estructura del proyecto tiene Next.js en subdirectorio `Web/`, lo que confunde a Vercel

## Estado del Proyecto
- Proyecto Next.js en: `Web/`
- package.json solo existe en `Web/`, NO en la raíz
- Git ignora archivos `*.json` en la raíz (excepto vercel.json y Web/package.json)

## Solución Paso a Paso

### Paso 1: Limpiar el archivo package.json de la raíz
- Verificar que el archivo `package.json` en la raíz solo contenga JSON válido
- El archivo debe tener exactamente:
```json
{
  "name": "lupa",
  "version": "1.0.0",
  "private": true,
  "scripts": {
    "dev": "cd Web && npm run dev",
    "build": "cd Web && npm run build",
    "start": "cd Web && npm start"
  }
}
```

### Paso 2: Actualizar .gitignore para incluir package.json de la raíz
- Agregar `!package.json` después de `!vercel.json` en el .gitignore
- Esto permitirá que git rastree el archivo

### Paso 3: Configurar Vercel correctamente
- **Root Directory**: `.` (punto - la raíz del repositorio)
- **Build Command**: `cd Web && npm install && npm run build`
- **Output Directory**: `.next`
- **Install Command**: `cd Web && npm install`

### Paso 4: Agregar Environment Variables en Vercel
Necesarias para que Firebase Admin funcione:
- `FIREBASE_PROJECT_ID` = `innergy-a55ba`
- `FIREBASE_ADMIN_CLIENT_EMAIL` = `firebase-adminsdk-fbsvc@innergy-a55ba.iam.gserviceaccount.com`
- `FIREBASE_ADMIN_PRIVATE_KEY` = (la key completa con saltos de línea reales)

### Paso 5: Redeploy
- Hacer redeploy después de configurar todo

## Configuración Final Esperada en Vercel

**Settings → General:**
- Root Directory: `.`

**Settings → Build & Development:**
- Build Command: `cd Web && npm install && npm run build`
- Output Directory: `.next`
- Install Command: `cd Web && npm install`

## Comandos a ejecutar localmente después de los cambios
```bash
git add -f package.json .gitignore
git commit -m "Fix package.json for Vercel deployment"
git push
```
